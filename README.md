# game-1
** Հիշի՛ր թե կարող ես
** 
Հիշողության և ուշադրության խաղ

Մասնակիցները հերթով հաշվում են՝ 1, 2, 3, 4, 5... մինչև 11։

Ամեն ինչ պարզ է — մինչև առաջին կանոնը։

Հենց որ ինչ-որ մեկը հասնում է 11 թվին — նա պարզապես «11» չի ասում, այլ հորինում է նոր կանոն ամբողջ խաղի համար։ Օրինակ՝

«7 թվի դեպքում ասել "գաֆ"»
«Բոլոր զույգ թվերը — "բում"»
«3-ից 5 թվերը — "հոպ"»
Դրանից հետո կանոնները բոլորին ցույց են տրվում 5 վայրկյան — հիշեք։ Հետո խաղը նորից սկսվում է, բայց արդեն նոր կանոնով։ Ամեն փուլի հետ կանոնները ավելանում են, և բոլոր կանոնները հիշելն ավելի ու ավելի դժվար է դառնում։

Քո հերթի ժամանակ դու չես տեսնում, թե հիմա որ թիվն է — միայն հարցական նշան։ Ինքդ պետք է հիշես, թե հիմա ինչ պետք է լինի, և ստուգես՝ արդյոք այն ընկնում է որևէ կանոնի տակ։

Սխալվեցիր — ասեցիր սխալ բառ կամ սխալ թիվ — դուրս ես գալիս։ Մնացածները շարունակում են առանց քեզ։ Հաղթում է վերջինը, ով բոլոր կանոնները հիշել է։
